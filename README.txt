Tikslas:
Pagilinti Java zinias.

Uzduotis issigalvota:
Naudojantis pure java metodais sukurti HTTP REST'api serverio MVC veikima imituojantcia programa
1. galincia uplodinti konkretu csv faila,
2. ji issaugti improvizuotoje db,
3. pagal requesta gauti JSON pavidalu atsakyma su paskaiciavimais.
4. parasyti paprasta front end'a galinti atvaizduoti JSON rezultatus.

Veikimas:
1. paleisti programa,
2. prisijungti prie http://localhost:8080 puslapio,
3. suaploadinti file.csv faila (sakniniame kataloge),
4. http://localhost:8080/result gauti JSON faila nusiusti duomenys,
5. http://localhost:8080/resultcaliculated gauti JSON faila paskaiciuoti duomenys,
6. http://localhost:8080/table HTML/JS rezultatu lenteles,
   JSON'u atvaizdavimas paimant is auksteniu (4,5 punktai) nuorodu.
